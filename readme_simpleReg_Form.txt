Overview
This is a simple Python-based graphical user interface (GUI) application for a registration form. It allows users to enter their personal details and displays the entered information upon submission.

The application is built using the tkinter library, which is included with Python. This makes it easy to use and portable across different systems without needing additional dependencies.

Features
User Input Fields:

First Name
Last Name
Email
Age
Password (hidden entry)
Gender (selectable via combobox)
Responsive Design:

The form dynamically adjusts its layout and accommodates various screen sizes.
Color Customization:

The application window is styled with a purple background.
Labels are styled with purple text.
Submitted data is displayed in blue text.
Submit Button:

Red-colored text for improved visibility.
Clicking the button triggers data validation and displays the submitted information below the form.
Requirements
Python 3.x
Tkinter (included with Python standard library)
